Worked Example 1

\[ \int_0^1 x^2 dx \]

Step 1: Compute Simpson's rule on [0,1] \[ h = \frac{1-0}{2} = 0.5 \] \[ m = 0.5 \] \[ S(0,1) = \frac{0.5}{3}[f(0) + 4f(0.5) + f(1)] \] \[ = \frac{0.5}{3}[0 + 4(0.25) + 1] = \frac{0.5}{3}[2] = 0.333333 \]
Step 2: Compute Simpson's rule on [0,0.5] \[ h = 0.25, m = 0.25 \] \[ S(0,0.5) = \frac{0.25}{3}[f(0) + 4f(0.25) + f(0.5)] \] \[ = \frac{0.25}{3}[0 + 4(0.0625) + 0.25] = 0.0416667 \]
Step 3: Compute Simpson's rule on [0.5,1] \[ h = 0.25, m = 0.75 \] \[ S(0.5,1) = \frac{0.25}{3}[f(0.5) + 4f(0.75) + f(1)] \] \[ = \frac{0.25}{3}[0.25 + 4(0.5625) + 1] = 0.291667 \]
Step 4: Error estimate \[ \text{error} = \frac{|S(0,1) - (S(0,0.5) + S(0.5,1))|}{15} \] \[ = \frac{|0.333333 - 0.333334|}{15} \approx 6.7 \times 10^{-8} \]

Since error < tolerance, result is \(0.333333\)

Exact value: \(\frac{1}{3} = 0.333333\)

Worked Example 2

\[ \int_0^{\pi/2} \sin(x) dx \]

Step 1: Compute Simpson's rule on [0,π/2] \[ a = 0, b = \pi/2 \approx 1.5708, h = 0.7854 \] \[ m = 0.7854 \] \[ S(0,\pi/2) = \frac{0.7854}{3}[\sin(0) + 4\sin(0.7854) + \sin(1.5708)] \] \[ = \frac{0.7854}{3}[0 + 4(0.7071) + 1] = 1.00228 \]
Step 2: Refine subintervals \[ S(0,\pi/4) \text{ and } S(\pi/4,\pi/2) \] After recursion, the algorithm converges to:
Result: 1.000000
Exact value: \([-\cos(x)]_0^{\pi/2} = -\cos(\pi/2) + \cos(0) = 0 + 1 = 1\)
