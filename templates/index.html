What is Adaptive Simpson Quadrature?

Adaptive Simpson's rule is a numerical integration method that automatically adjusts the step size to achieve desired accuracy. It uses Simpson's rule recursively on subintervals where the error estimate is too large.

Mathematical Background

The Simpson's rule approximation for an integral is:

\[ \int_a^b f(x) dx \approx \frac{h}{3}[f(a) + 4f(m) + f(b)] \]

where \( h = \frac{b-a}{2} \) and \( m = \frac{a+b}{2} \).

The adaptive algorithm computes the integral on \([a,b]\) and compares it to the sum of integrals on \([a,m]\) and \([m,b]\). If the error estimate exceeds tolerance, it recursively refines the interval.

Error Estimate

The error estimate is given by:

\[ \text{error} \approx \frac{|S(a,b) - (S(a,m) + S(m,b))|}{15} \]
